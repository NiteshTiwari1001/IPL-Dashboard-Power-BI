# 🏏 IPL Analytics Dashboard (2008–2025) | Power BI

## 📌 Project Overview
This project is an interactive **IPL (Indian Premier League) Dashboard** built using Power BI, covering seasons from **2008 to 2025**.

The dashboard provides deep insights into team performance, player statistics, match outcomes, and season-wise analysis using advanced **DAX measures and data modeling**.

---

## 🎯 Key Features

### 🔍 Season-Based Dynamic Analysis
- Fully interactive **Season Filter**
- All KPIs and visuals dynamically update based on selected season

---

## 🏆 Primary KPIs
- 🥇 **Champion (Winning Team)**
- 🥈 **Runner-Up Team**

---

## 📊 Secondary KPIs
- 🏟️ **Total Matches Played**
- 🌍 **Total Venues**
- 👥 **Total Teams Participated**
- 🔥 **Total Fours**
- 💥 **Total Sixes**
- 💯 **Centuries**
- 5️⃣0️⃣ **Half-Centuries**

---

## 🧢 Player Performance Insights

### 🟠 Orange Cap (Top Run Scorer)
- Player with highest runs in selected season
- Displays total runs scored

### 🟣 Purple Cap (Top Wicket Taker)
- Player with most wickets in selected season
- Displays total wickets taken

---

## 💥 Batting Highlights
- Player with **Most Fours in a Season**
- Player with **Most Sixes in a Season**

---

## 📈 Points Table
- Dynamic **Points Table** for selected season
- Includes:
  - Matches Played
  - Wins / Losses
  - Points
  - Team Rankings

---

## 🧠 Data Modeling
- Multiple tables used:
  - Matches Data
  - Deliveries / Ball-by-ball Data
  - Teams Data
  - Players Data

- Relationships created for:
  - Season filtering
  - Team-level aggregation
  - Player-level statistics
